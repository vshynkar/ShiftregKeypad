# ShiftregKeypad


![alt tag](https://github.com/vshynkar/ShiftregKeypad/blob/master/74HC165_pinouts.png)

Keypad connector pins:
* 1 - GND
* 2 - VCC
* 3 - Q7 inverted (7 pin on chip)
* 4 - SCK (2 pin on chip)
* 5 - PL (1 pin on chip)
* 6 - INT (connect to interruption pin on arduino)
